# 事後檢定

```sas
proc GLM data = Hedata;
	class LIV_ABSO;
	model BW = LIV_ABSO;
	means LIV_ABSO / SCHEFFE;
run;
```

```sas
proc GLM data = Hedata;
	class LIV_ABSO;
	model BW = LIV_ABSO;
	means LIV_ABSO / DUNNETT ('0');
run;
```

```sas
proc npar1way data = Hedata wilcoxon;
    class LIV_ABSO;
    var BW;
run;
```

# Practice

```sas
data df;
	set Hedata;

	if HBA1C < 5.6 then PHBA1C = 0;
	else if 5.6 <= HBA1C < 6.5 then PHBA1C = 1;
	else PHBA1C = 2;

proc npar1way data = df wilcoxon;
	class PHBA1C;
	var BW;
run;
```
