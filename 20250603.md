# ANOVA

```sas
proc anova data = HEDATA;
	class Liv_ABSO;
	model LDL = Liv_ABSO;
run;
```
- class 組別
- model 變數 = 組別


# practice

```sas
data Hedata_DBP;
    set HEDATA;

    if DBP < 60 then CDBP = 1;
    else if 60 <= DBP < 90 then CDBP = 2;
    else if 90 <= DBP then CDBP = 3;

proc anova data = Hedata_DBP;
    class CDBP;
    model LDL = CDBP;
run;
```
