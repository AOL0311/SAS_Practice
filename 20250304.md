```SAS
libname DEF 'D:\';

```

# 匯入 excel 檔案

```SAS
proc import datafile='C:\DOWNLOAD\Stu_heig.xlsx' out=df dbms=xlsx replace;
getnames=yes;
run;
```

### import 教學

```SAS
proc import datafile='檔案位置' out='匯出名稱'
dbms=excel replace      /* 辨識檔案格式 */
range='';               /* 選擇讀取的 excel 工作表 */
getnames=yes;
run;
```
